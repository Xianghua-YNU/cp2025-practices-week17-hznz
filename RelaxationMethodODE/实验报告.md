### 2.1 算法思路

使用二阶中心差分近似二阶导数，将微分方程转化为差分方程，然后通过松弛迭代法求解边值问题。

```python
# 计算稳定性参数c

delta = float('inf')
iteration = 0

while delta > tol and iteration < max_iter:

    # 计算最大变化量

    iteration += 1
```
